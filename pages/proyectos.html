Proyectos

Se realizaron diversos proyectos y servicios de asistencia para industrias alimenticias, especialmente para ingenios azucareros. Algunos ejemplos de esto son el diseño y construcción de tableros de control para tolbas, balanzas y paletizadoras.

También programación de control sobre Transportes de líneas, revampings de sistemas, y servicios de asistencia para modificaciones en programas de PLC y Paneles. Nuestros principales clientes se encuentran radicados en la provincia de Tucumán

En este proyecto se diseño un tablero que permitió un monitoreo preciso de todo el proceso de clasificación de semillas. El tablero con un sistema integrado, permite obtener información en tiempo real de todos los dispositivos que forman parte de dicho proceso, ya sea motores, transportes, válvulas, y demás.

Con esto se logra centralizar y mejorar el acceso a información útil para aumentar los niveles de productividad. Se trabajó con un PLC Allen Bradley CompactLogix y software SCADA FactoryTalk View.

Se realizaron distintos proyectos basados en configuración de RTUs. Estos equipos permiten comunicarse con diferentes tipos de dispositivos de medición y control dentro de centrales de generación y distribución electrica, como Seccionadores, Interruptores, Transformadores de Tensión e Intensidad. También permiten enviar toda la información concentrada a un sistema SCADA, logrando gestionar y monitorear todos los dispositivos en una central de operación.

Se trabajó con protocolos de comunicación Modbus RTU, Modbus TCP, IEC 61850, DNP3, PROFIBUS, PROFINET.

Se realizó la instalación de medidores bidireccional monofásicos, para darle la posibilidad que algunos usuarios inyecten energía a la línea de distribución.

A su vez, todos los medidores del edificio están conectados a un Concentrador, lo cual le permite a los lecturistas de las empresas distribuidoras, con solo conectarse a este mismo, obtener todas las lecturas de TODOS los medidores. Antes debían leer uno por uno, con este proyecto se reduce el tiempo de lectura y aumenta la eficiencia de los mismos.

Se realizó la provisión de una celda de palletizado para baldes de pintura de 4 y 20 litros. El proyecto incluyó la colocación de un robot de última generación KUKA KR180 con sus respectivas base y grippers, también los transportes de alimentación de baldes, provenientes de una linea de envasado y un dosificador automatico de pallets de madera (Con sus respectivos transportes).

El sistema de control se basó en un PLC CompactGuardlogix, un panel PanelView+ y una Caja de Acceso Multifuncional MAB 442G, todo el sistema cumple con un nivel de seguridad SIL 2.

En este proyecto se realizó la automatización de una planta de Hormigón Elaborado con el fin de ser operada, desde una cabina de control, por un único operario.

El sistema instalado permite seleccionar y modificar el tipo de receta a preparar como también gestionar cada uno de los dispositivos que forman parte de la planta.

Por otro lado, luego de finalizar la carga de cada camión, se genera un remito de manera automática donde se muestra un detalle de los datos del cliente, los consumos de cada material, y el tipo de receta utilizada.
